# Lab 2 - Landing page

Create a web page for your/friends/relatives business (example: car-rent, cookies-shop, crypto-scam)

> For a maximal grade, the website should be for a real client (your/friends/relatives business), otherwise -1-2 points

## Customer requirements

- Some form of navigation to sections (links, buttons)
- Call to action
- At least 4 sections ([see examples](https://webflow.com/blog/high-converting-landing-page))
- Pleasant to the eyes

## Dev requirements

- Use vanilla CSS and HTML (no frameworks)
- Have a decent git history (no one commit)
- The page should be deployed on a free hosting service (example: GitHub Pages, Vercel, Netlify)
- Project repo should have a README with a short description of the landing page topic, screenshots, and a link to the live demo

## Project Description

**Silver Valley** is an elegant landing page for a premium silver jewelry business. The website showcases a beautiful collection of handcrafted 925 sterling silver jewelry and accessories, including rings, earrings, brooches, and necklaces. The landing page features a modern, responsive design with smooth navigation, highlighting the brand's commitment to quality craftsmanship and timeless elegance.

The page includes four main sections:
- **Hero Section**: Eye-catching introduction with call-to-action buttons and key statistics
- **About Section**: Brand story and key features (craftsmanship, quality materials, unique designs, customer satisfaction)
- **Collections Section**: Showcase of 10+ unique silver jewelry pieces with pricing
- **Contact Section**: Contact information and social media links
